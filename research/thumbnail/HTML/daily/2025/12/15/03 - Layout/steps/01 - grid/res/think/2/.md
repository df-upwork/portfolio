Вот предложения по правкам кода для реализации верстки на `display: grid`.

# 1. Изменение селектора `body`

В этом блоке добавляется объявление сетки с двумя колонками (фиксированная для сайдбара и автоматическая для контента) и одной строкой, занимающей всю высоту контейнера.

```css
body {
	background-color: #ffffcc;
	display: grid;
	font-family: 'Carlito';
	grid-template-columns: 38.28px 1fr;
	grid-template-rows: 1fr;
	height: 800px;
	margin: 0;
	overflow: hidden;
	padding: 0;
	width: 1000px;
}
```

# 2. Изменение селектора `#sidebar`

Удаляются свойства абсолютного позиционирования и явные размеры, так как теперь элемент автоматически занимает первую ячейку сетки и растягивается на всю высоту.

```css
#sidebar {
	background-color: #ff6133;
}
```

# 3. Изменение селектора `#content`

Добавляется свойство `align-self: center` для вертикального выравнивания содержимого по центру ячейки сетки.

```css
#content {
	align-self: center;
	line-height: 1.1;
	text-align: center;
	white-space: nowrap;
}
```
